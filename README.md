# Interview Monorepo

This repository contains exercises and notes for interview preparation.

## Top-level Structure

- `foundations/`
- `roles/`
- `roadmap/`

## Directory Purpose

- `foundations/`: Core topics such as algorithms, programming languages, design patterns, and system fundamentals.
- `roles/`: Role-oriented interview preparation (for example: frontend, backend, mobile, product, and QA).
- `roadmap/`: Learning plans, milestones, and progress tracking.

## Current Status

The repository is organized under this structure and will continue to grow over time.
